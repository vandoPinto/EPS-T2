VIII. Descrever o planejamento dos encontros

O planejamento específico de cada encontro deverá contemplar a programação das atividades, a descrição da sequência didática, as leituras indicadas e a seleção dos materiais e dos recursos a serem utilizados. Na educação, chamamos esse documento de plano de aula.

Clique nos ícones e acompanhe um exemplo para a organização dessas informações (GOES et al., 2015; TOBASE et al., 2021).

PROGRAMAÇÃO

Deve descrever a sequência didática do encontro e os horários para cada atividade.

x

BIBLIOGRAFIA

Devem ser selecionadas as referências e os materiais de leitura para o encontro. Pode ser indicado para antes do encontro, como leitura prévia, ou após, para aprofundamento dos estudos.

x

AVALIAÇÃO

Deve acontecer de forma processual e contínua, pois também faz parte do processo de aprendizagem.

x

RECURSOS DIDÁTICOS

Devem ser definidos os meios necessários para a realização das estratégias de aprendizagem. Podem ser recursos físicos (lousa, flipchart, tarjetas) ou tecnológicos (internet, computador, celular).

x

ESTRATÉGIAS DE APRENDIZAGEM

Devem ser selecionadas as estratégias utilizadas em cada etapa do encontro.

x

CONTEÚDO

Deve ser elaborado de forma a alcançar o objetivo de aprendizagem proposto para o encontro.

x

OBJETIVO DE APRENDIZAGEM

Deve ser elaborado de forma a contribuir com o cumprimento do objetivo geral definido para a formação.

x

DADOS GERAIS

Devem ser definidos o título do encontro, a data, o horário, o local, a carga-horária e o facilitador de aprendizagem.
